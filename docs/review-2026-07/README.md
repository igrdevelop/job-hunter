## Рекомендуемый порядок

**A (петля обратной связи) → B (качество) → D (ops) → C (долг) → E (расширение).**

Логика: инфраструктура генерации сейчас сильнее, чем понимание её отдачи.
Система умеет делать хорошие CV дёшево ($0.13–0.17), но по-прежнему не знает,
какие источники/треки/скоры приводят к ответам работодателей. Данные для этого
уже копятся (funnel, verdict в колонке N, confirmations) — путь A превращает их
в решения и стоит дешевле всех остальных путей.
